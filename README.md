# 🏦 Banking Loan Portfolio Analysis

## 📌 Overview
Developed an interactive Power BI dashboard to analyze loan portfolio performance, customer demographics, branch performance, and repayment trends. The project helps monitor key lending KPIs and supports data-driven business decisions.

## 🛠️ Tools Used
- Power BI
- SQL
- Excel
- Power Query
- DAX

## 📊 Key KPIs
- Total Loan Amount
- Active Loans
- Repayment Rate
- Default Rate
- Total Borrowers
- Branch Performance
- Loan Disbursement Trend

## 💡 Key Insights
- Analyzed branch-wise loan performance and repayment efficiency.
- Identified regions with higher default rates.
- Monitored borrower demographics and loan portfolio trends.
- Built interactive dashboards using DAX, Power Query, and SQL.
